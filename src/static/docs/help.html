#Top Navigation #Side Navigation #Modules #Settings
Top Navigation

Left side - tabs for each module. Navigate between modules by click on each tab or using the keyboard shortcuts.
Right side - icon executes the 'quick connect' feature. The icon opens the settings modal. Finally, the icon opens keyboard shortcut maintenance.

Side Navigation

Holds external links, which are customizable by clicking the 'pencil' icon found at the bottom of the side navigation. An 'Email' link will always show as the first option within the first section. Currently only Gmail is supported. The side navigation is limited to four sections with 8 links each.

Modules

Current modules - Customers, Time, Snippets
Adding new record - new records can be entered by clicking the '+' button found on the bottom right corner of each module or using the available keyboard shortcuts.
Editing record - records can be maintained by click the 'pencile' icon found next to each record.

Settings
Section Description
Connection File Path - path in OS that the connection files reside. (.tnt files)
Quick Connection - connection file to be used for the 'quick connect' feature. Gets opened using 'connect' icon within the navigation bar.
Data Download Data - exports specified table's data into .json files to download.
Upload Data - upload data in format of json into specified table.
Clear Data - clears specified table's data
Modules Customers - enable/disable the customers module. Data will not be wiped if disabled.
Time - enable/disable the time module. Data will not be wiped if disabled.
Snippets - enable/disable the snippets module. Data will not be wiped if disabled.
